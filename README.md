# CPSC 362 Finance App

A Python Tkinter desktop app for tracking shared expenses, balances, and payments.

## Run

```powershell
python main.py
```

## Project Structure

- `main.py` starts the app.
- `gui.py` contains the Tkinter interface.
- `models.py` contains data models and validation.
- `storage.py` handles local JSON files.
- `balance_manager.py` calculates balances and settlement suggestions.
- `data/` stores app data in JSON files.
